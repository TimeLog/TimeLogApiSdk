# TimeLog New Tab Extension Beta

This extension is made by Søren Øxenhave, solution architect at TimeLog A/S. 

SUPPORT
The extension is in a beta version and is not supported by TimeLog A/S.

OPEN SOURCE
The extension is open sourced and the complete source code is available from Github as part of the TimeLog API SDK via the official extension website.
